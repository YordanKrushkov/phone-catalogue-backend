<h4>Description</h4>
A back-end for a single page application that allows the users to browse for mobile phones. The logged users can delete/edit the adverts.

<h4>Technologies</h4>
Node.JS, Express, Mongo DB, Mongoose

<h4>API's</h4>
Cloudinary
